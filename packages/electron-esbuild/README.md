## Getting Started

With npm:

    npm init @electron-esbuild/app

With yarn:

    yarn create @electron-esbuild/app

**All configurations are already setup for you.**

Start a development build

```shell
npm run dev
```

Create a build

```shell
npm run build
```

```shell
npm run build -- --no-clean # do not clean output before build
```

Package the app

```shell
npm run package
```
